# SNESToGameport

Converts an SNES controller to a Gameport.

SNES wiring:    
    Power - 5V    
    Latch - PA4    
    Clock - PA5    
    Data - PA6    

Gameport Axis Schematic:    
    ![Axis Schematic](https://github.com/netham45/SNESToGameport/blob/main/axis_schematic.png?raw=true)
     

Gameport wiring:    
    X1 Center  - PB0    
    X1 Max     - PB1    
    Y1 Center  - PB2    
    Y1 Max     - PB3    
    X2 Center  - PB15    
    X2 Max     - PC15    
    Y2 Center  - PB6    
    Y2 Max     - PB7    
    Button 1   - PB10    
    Button 2   - PB12    
    Button 3   - PB13    
    Button 4   - PB14
